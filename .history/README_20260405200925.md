# Shape Up The Vote — Data Explorer

An interactive Streamlit app built on top of the SUTV 2024 voter engagement data pipeline.

## Pages

### 1. State Overview
Salon & barbershop density across 13 swing states, benchmarked against the national average (1 salon per 773 people). Filter by Wave 1 / Wave 2 launch states.

### 2. Service Density KPI Calculator
Input any city's population and scraped shop count to calculate the Service Density KPI — a normalized 0–1 score measuring how well our dataset covers that locality relative to its state benchmark. Displays a live comparison against other localities in the same state.

### 3. Zip Code Prioritization
Filter zip codes by state, priority flag, median household income, and non-white population percentage. Based on the hypothesis that communities of color and lower-income areas are underserved in voter engagement efforts.

## Running Locally

```bash
uv venv
source .venv/bin/activate  # Windows: .venv\Scripts\activate
uv pip install -r requirements.txt
streamlit run app.py
```

## Data Note

This app uses a synthetic dataset that mirrors the schema of the original pipeline. Real zip codes and city names are used; demographic values are plausible approximations drawn from Census distributions. The original pipeline connected to BigQuery (`sutv.*` tables) and Google Cloud Storage.

## Original Pipeline

- **Data collection:** Pline.io Chrome extension → Google Places
- **Cleaning & enrichment:** Hex (data notebook) + US Census API
- **Analysis:** SQL (BigQuery) + Python (pandas)
- **Output:** `high_priority_zips.csv` → Google Cloud Storage
